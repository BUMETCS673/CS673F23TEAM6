# collegestreet web-app code

## Tech Stack

 - ReactJS (UI framwork)
 - TailwindCSS (Styling)
 - Websocket (for Chat feature)
 - cypress (For testing)

## Steps to run Localhost
 1. Open terminal
 2. Go the the code/web-app folder
 2. Install all the dependencies
 `npm install`
 2. Run
 `npm start`
 4. View in your browser
